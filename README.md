# linux-commands-Pig-and-hive-queries
📌 Pig and Hive Queries – Simple Notes

This repository contains basic Pig and Hive queries explained in a very simple and easy-to-understand way using a small student dataset.

The goal of this file is to help beginners quickly understand how Pig Latin and Hive SQL work without getting confused by complex syntax. Each query is written step-by-step with a clear explanation of what it does and why it is used.

👨‍💻 What’s included:

Sample dataset (students.csv)

Basic Pig queries (LOAD, FILTER, GROUP, FOREACH, STORE)

Basic Hive queries (CREATE, SELECT, WHERE, GROUP BY, ORDER BY)

Simple explanations for each query (like class notes)

📊 Dataset used:
Includes sample student data with names like gagan, rohan, reka, sunil, and anitha.

Useful for lab exams

Quick revision before viva

Easy understanding for beginners
